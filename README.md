# Rock Paper Scissors 🎮

Python command-line game with ASCII art.

## Play
```bash
python game.py
```
Enter `0` (Rock), `1` (Paper), `2` (Scissors).

## Features
- ASCII art hands
- Random computer choice  
- Win/lose/draw logic
- Input validation
